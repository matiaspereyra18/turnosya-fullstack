# 🗓️ TurnoYa - Sistema de Gestión de Turnos y Citas Médicas

![React](https://img.shields.io/badge/Frontend-React.js-61DAFB?style=for-the-badge&logo=react&logoColor=black)

![MySQL](https://img.shields.io/badge/Database-MySQL-4479A1?style=for-the-badge&logo=mysql&logoColor=white)

**TurnoYa** es una aplicación web Full Stack diseñada para la reserva y administración de citas o turnos en tiempo real. Permite registrar, listar y cancelar reservas, sincronizando los datos de manera persistente con una base de datos relacional.

---

## 🚀 Tecnologías Utilizadas

* **Frontend:** React.js, JavaScript (ES6+), HTML5, CSS3, Fetch API / Axios.
* **Backend:** Node.js, Express.js, APIs RESTful, CORS, Dotenv.
* **Base de Datos:** MySQL, phpMyAdmin / XAMPP.

---

## 🛠️ Arquitectura de la Aplicación

1. **Frontend (React):** Interfaz dinámica que interactúa con el usuario y realiza peticiones HTTP (`GET`, `POST`, `DELETE`) al backend.
2. **Backend (Node.js + Express):** Servicio en encargado de procesar las solicitudes, ejecutar la lógica de negocio y comunicarse con la base de datos.
3. **Persistencia (MySQL):** Base de datos relacional que almacena las entidades y la información de los turnos.
